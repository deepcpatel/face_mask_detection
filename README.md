# Face Mask Detection
Face mask detection in PyTorch.

**Platform**: PyTorch 1.1.0, OpenCV 4.1.1, Ubuntu 18.04.

**Dataset**: [RMFD dataset](https://github.com/X-zhangyang/Real-World-Masked-Face-Dataset) [2].
